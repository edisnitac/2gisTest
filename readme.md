Приложение для анализа частоты слов в текстовых файлах с визуализацией результатов.

Логика работы:

1. Пользователь выбирает файл через интерфейс( кнопка `Старт` вызывает файловый диалог с фильтром на текстовые файлы) 
2. `FileProcessor` читает файл порциями (чанками)
3. `WordCounter` обрабатывает текст и подсчитывает слова
4. Результаты передаются в QML через `AppController`
5. Интерфейс отображает прогресс и гистограмму

Основные компоненты:

1. C++ Модули:
	1. `AppController` - Мост между C++ и QML
	2. `FileProcessor` - Чтение файла порциями, управление потоком обработки 
	3. `WordCounter` - Подсчет частоты слов, формирование топ-слов

2. QML Компоненты:
	1. `MainWindow` -  Главное окно приложения  
	2. `HistogramCustom` - Гистограмма для визуализации топ-слов 
	3. `ProgressBarCustom` - Прогресс-бар
	4. `ControlPanel` - Панель управления (старт/пауза/отмена)
	
Сборка и запуск:

Собиралось и запускалось с помощью Qt 5.15.2 MSVC2019 64bit и qmake( из этого следует что Windows, в моем случае 10ая)
